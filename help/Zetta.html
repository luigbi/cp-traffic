Zetta

The Zetta automation export and import is enabled on the Traffic Site Options screen, Automation tab, by checking on the “Zetta” checkbox.


With the Zetta automation export and import, custom export and import folders can be specified on the Vehicle Options screen Log tab. If custom folders are not defined, the default export and import folders will be used.
