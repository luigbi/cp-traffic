Restrictions

The fields on the Restrictions tab are used to set additional user options for each user. Green means the user has full read/write access, and red means they do not have access to that feature. (The view only option is not available for any of the features controlled by this screen except for the Activity Log.)


The Restrictions tab has the following sections: Alerts, Utility, General, Agreement and Comment. (The order of these buttons is slight different on versions prior to version 7.0.)

Alerts

A user can be set to receive or not receive an alert by changing the Alert to be red or green in the Alerts section. 


  • Export Spots: Alerts the user that logs have been generated in the Traffic System and are ready to be exported to the Counterpoint Affidavit system.
  • Export ISCI: Alerts the user that logs have been generated in the Traffic System and the ISCI Export is ready to be generated.
  • Traffic Reprint Logs/CPs: Alerts the user that the log has been reprinted on the Traffic system. This alert provides a notification of the need to re-export.
  • Allowed to Initiate Shutdown: When set to green, this user will have the ability to perform a system shutdown, so that maintenance can be performed or for other reasons. For more information on the System Shutdown feature, please see the help document “IT Utilities” that is available on the Counterpoint website.
  • Web Vendor: On version 7.0 and above, if the “Web Vendor Manager” task is enabled for monitoring using the Task Monitor, a “Web Vendor” alert button will be shown. When set to Green, the web vendor alert will be available to view on the Affiliate Alerts screen (when there's an alert, a red Alerts button will be present at the top of the screen). When this alert restriction is set to Red, the web vendor alert will not be visible, and the radio button to select it from the Affiliate Alerts screen will not be present.

Utility

  • Clear Posted Spots: Determines whether the user can run the Clear Posted Spots utility without having to input a password obtained from Counterpoint support.
  • Delete Posted Spots: Determines whether the user can run the Delete Posted Spots utility without having to input a password obtained from Counterpoint support.

General

  • Activity Log: Determines whether the user has the “Show Activity Log” checkbox on the Accessories -> User Status screen. 
  • Override Export Queue Priority: Determines whether the user can change the priority of exports in the Export Queue.
  • Export Specifications: Determines whether the user can save changes on the  Specifications in the Export Center and the Specifications screen available from the Export menu.

Agreement

  • Define Pledge Information: Determines whether the user can define pledges on the agreement screen.

Comment

  • Change: Determines whether the user can change a previously entered comment on the Affiliate Management screen.
  • Delete: Determines whether the user can delete a previously entered comment on the Affiliate Management screen.
