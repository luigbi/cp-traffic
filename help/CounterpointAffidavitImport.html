Counterpoint Affidavit Import

Posted spots can be imported from the Counterpoint Affidavit System automatically or manually.

Automatic Import

There are two ways to configure the automatic import: the BAT file method or the executable method.

The BAT file method

To set up the automatic import to run using the BAT file method, open the CSI_Server.ini file using a text editor and edit the settings as follows:


Add “AutoImport” to the Main Settings area in the SectionNames field. (Each item in the SectionNames field must be separated by a comma as shown in the example below.)


[MainSettings]

SectionNames = Backup, AutoImport


Add the AutoImport section to the CSI_Server.ini, basing it on the example below. Edit the drive letters and paths to match the system configuration.


[AutoImport]

Enabled = 1

Weekdays = 1111111

Interval = 86400

StartTime = 3:00am

MonthDaysToRun = 

TimeSlot =

Exe = c:\csi\prod\exe\startAffiliateImport.bat

StartInFolder = 

CommandLine = 

KeepProgramRunning = 0


Note: Set the StartTime to when the auto import should run. 


After editing the CSI_Server.ini file, locate the line that starts with “LastUpdated” and remove the date and time after the equal sign, and then save the file. This will cause the service to reload the new settings.


The next step is to create a BAT file with the following contents:


REM the next line must be changed to match the customer’s server name 

Net Use M: \\SERVERNAME\CSI

REM Change to the M drive

M:

REM Set the working folder

CD\

CD prod\data

REM Start the import process

M:\prod\exe\Affiliat2.exe autoimport

REM Pause


Edit the server drive letter as needed to match the system configuration, then save the BAT file in the csi\prod\exe folder.

The executable method

The automatic import can also be configured to run as an executable, without the use of a BAT file.


To set up the automatic import to run using the executable method, open the CSI_Server.ini file using a text editor and edit the settings as follows:


Add “AutoImport” to the Main Settings area in the SectionNames field. (Each item in the SectionNames field must be separated by a comma as shown in the example below.)


[MainSettings]

SectionNames = Backup, AutoImport


Add the AutoImport section to the CSI_Server.ini, basing it on the example below. Edit the drive letters and paths to match the system configuration.


[AutoImport]

Enabled = 1

Weekdays = 1111111

Interval = 86400

StartTime = 3:00am

MonthDaysToRun = 

TimeSlot =

Exe = c:\csi\prod\exe\affiliat2.exe

StartInFolder = c:\csi\prod\csiutils

CommandLine = autoimport

KeepProgramRunning = 0


Note: Set the StartTime to when the auto import should run. 


After editing the CSI_Server.ini file, locate the line that starts with “LastUpdated” and remove the date and time after the equal sign, and then save the file. This will cause the service to reload the new settings.


If it hasn’t already been done, create a CSIUtils folder in c:\csi\prod and copy the Traffic.ini and Affiliat.ini to the new folder. Edit the drive letters for all the paths on the two .ini files from the mapped drive letter to the local server drive letter (for example, m:\csi\prod\exe should become c:\csi\prod\exe).

Manual Import

To import posted spots from the Counterpoint Affidavit System, go to File -> Import -> Counterpoint Affidavit, and click Import. 


You will see the import’s progress in the white Results box as it gathers aired information. 


The date span of imported aired information is determined by Site settings. 


Once completed, all results will appear in the Results box, and a web import  log is created and can be viewed through the Accessories>Messages viewer. The file is stored in the CSI\Prod\Import folder, and named WebSpots_SARA-PC_SaraSmith_140103_112345 (Filename_computer name_user name_yymmdd_hhmmss).
