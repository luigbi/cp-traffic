Enco

The ENCO export creates one export record for each day, time zone (time zones marked with an *) and vehicle, along with a PDF log for verification of the export files. All time zones are combined in this report which contains the feed time (adjusted by the vehicle zone table), advertiser/product, spot length, ISCI, creative title and time zone. Export records are stored in the CSI\Prod\Export.

Running the Export

To run the export, go to Task Menu -> Export -> Enco. 


Enter a single date or a span of dates.  No end date assumes same as start date.

Select the vehicles, and click Export.

Any missing copy, creative titles, or invalid advertiser names are shown in the error message box to the right of the vehicle list. These messages are stored in the ExptEnco.txt and can also be viewed and printed via the Accessories- > Messages Viewer.

File Information
