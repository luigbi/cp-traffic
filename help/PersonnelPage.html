Personnel Page

When clicking the Personnel button at the top of the page, the Personnel page will be displayed. This is a list of station personnel user accounts for the logged in station.



The following information is shown:


  • Email address
  • First name
  • Last name
  • Title (non-mandatory)
  • Phone (non-mandatory)
  • Action: the following actions are available:
    • Edit: edit the user account information.
    • Resend password reset link: clicking this button will send the user an email with a link they can click on to reset their password. 
    • Delete: delete the user account. 


Note that a logged in user cannot resend themselves a new password link or delete their own account. These buttons will not be present for a logged in user's own account.


Station users have the ability of adding new station personnel for their station.

To add a new station personnel user account:


  1. Click the Add User button in the upper right corner.
  2. Enter their first name, last name, and email address.
  3. Optional: enter their job title and phone number.
  4. Press Save. This will trigger an automated email that gets sent to the new user's entered email address. The new user must then click the link and create a password to complete the process of creating their account. Note that the link expires in 24 hours. If they have not confirmed their new account within 24 hours, the "Resend password reset link" can be used to send them a new link.


When clicking a valid email confirmation link, the new user will be presented with an account activation page, as shown below.



To activate their new account, they must enter a password that meets the stated requirements, and a confirmation password that matches the password, and then press the Confirm Password button. When successful, they will then be able to log in using their newly validated account.


Important note: a single email address can be associated with multiple stations. When this is the case, they will use the same login information for each station, and can switch between their associated stations using the station dropdown at the top of the Dashboard page.
