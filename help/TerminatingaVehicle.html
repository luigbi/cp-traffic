Terminating a Vehicle

Follow these steps to terminate a vehicle and the programming. 

 

  1. Run the Paperwork Summary Report to obtain a list of contracts referencing the vehicle that is to be terminated.
    1. Enter active start date as the Monday after the date that the vehicle will be terminated 
    2. Leave all other dates as blank.
    3. Select the report by Vehicle Option
    4. Select the vehicle(s) to cancel/remove from the list
    5. Include all contract types (add Remnant, PSA, and Promo)
    6. Leave all other values on the screen
    7. Generate the Report
  2. Terminate the schedule lines for each Contract as of the last date that the vehicle will be airing
  3. Schedule the changed contracts
  4. Run paperwork summary again, with the same values as in Step 1
    1. If any contracts are reported, then repeat Step 2 for each contract that is reported
  5. Remove the vehicle from standard packages
  6. Remove the vehicle from each rate card that might be used for new and old changes to orders
    1. This step is required to eliminate the vehicle from showing on the proposal/order screen.  It will show on old orders that referenced the vehicle only
  7. If this is a Selling vehicle, the links from the termination date plus should be removed
    1.  If the termination date is far in the future, it is best to leave the links since the user would not be able to make any other link changes prior to the termination date
  8. Remove programming including the TFN dates
    1. Drag the Monday program image to the trash
    2. Enter the start date as one day after the last day the program is to air.
    3. Leave the end date as TFN
    4. Select each day as appropriate
    5. Schedule the program changes
    6. Check that all programming was removed from the day after the program is to last air including TFN.  This can be done visually on the programming  screen.  If the programming has not been removed, then repeat steps 6A â€“ 6F
  9. Mark the vehicle as Dormant via the Lists -> Vehicle screen
