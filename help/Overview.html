Overview

When launching the Proposal screen, it will first appear as shown in the picture below, with three main sections.


The top section is used to select the task you will be performing on the screen.


The header area is used to view or enter basic information about the proposal, such as the advertiser name and agency.


The line area lists the vehicles, dayparts, lengths, and spots that make up each contract line.

There are different views available in the Line area by pressing the tabs that appear at the top of the line area. The available tabs are: Multimedia, Digital Content, NTR, Air Time, Merchandising, Promotional, Installment, Avails, Protection, and Research. There are explanations of each tab in this document. Not all tabs will be available on all systems and for all user accounts.


There are buttons in the upper right corner and lower left corner of the screen that are used to perform various actions, such as generating the Proposal XML file or saving the proposal. There are explanations of each button in this document.

Selecting Tasks and Contracts

The top part of the screen is used to select what task you will be performing, such as adding a new proposal or editing an existing one, and for selecting the contract number when viewing or editing an existing contract.

Step 1: Select Task

To select the action you wish to do, click the “Step 1: Select Task” dropdown at the top of the screen, and select one of the available tasks.

 

The available tasks are as follows:


Create Proposal from Scratch: create a new proposal without modeling it from an existing contract.


Create a Proposal by Modeling: create a new proposal based on an existing contract. For example, when there’s a Quarter 1 contract that will be duplicated for Quarter 2, or when there’s a similar contract that exists that you wish to copy for the new proposal (and then possibly make changes to it).


Change an Existing Proposal: edit an existing proposal that has already been entered in the system and that needs to be revised.


Create New Version of Existing Proposal: create an alternate version of an existing proposal. The original proposal version will still exist after doing this, along with the alternate version that you create.


Revise an Existing Hold or Order: revise a hold or order, which is a proposal that has been approved and changed into a hold/order. Not all users will have access to this option.


Change Hold to Order: change the status of a hold contract to an order. Not all users will have access to this option.

View: view a contract in view-only mode, without changing it. This task can also be used to compare one version of a proposal to another, or one revision of an order to another.

Step 2 and 3: Select Advertiser and Number

Step 2 and 3 are used with any task other than “create a proposal from scratch” and “create a proposal by modeling” to select the advertiser and contract number that will be changed or viewed.


For example, to view a contract for “AC Delco”, first select “View” as the Step 1 task, press Tab to get to Step 2, click the advertiser dropdown arrow, and select “AC Delco” from the advertiser list.

After selecting the advertiser from the advertiser dropdown list, press Tab to get to Step 3 and click the dropdown arrow. Step 3 will show a list of available contracts for the selected advertiser, as shown in the picture below.

Click the contract you wish in the list to view to bring it up on the screen.
