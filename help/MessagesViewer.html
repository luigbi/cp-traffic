Messages Viewer


When certain tasks are performed, such as generating an export, or running logs, Counterpoint automatically creates a log text file that records information about the task. These files are saved to the Messages folder which can be accessed through Accessories -> Messages in the Traffic and Affiliate Systems. 


Select the file to view from the list and the information will appear in the box below.


To view a file, simply click on the file and the information will appear in the white box below. Certain files, Logs and Invoicing, require you to also select a user to view activity.

To view a file, simply click on the file and the information will appear in the white box below. 

Multiple Files

There are some log files that require additional selectivity. The Marketron and X-Digital Export Logs require the actual week to be selected.

Printing the Log File

You can print the information viewed, by clicking on the Printer icon in the bottom right corner.

Emailing the Log File

Press the Email button to email a log file. A message will appear that asks whether to include the log file as an attachment. Answer Yes to include the log file as an attachment.


Fill out the required fields then press Send to send the email.
