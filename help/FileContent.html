File Content

The X-Digital file name cannot exceed 32 characters. If the file name is longer than 32 characters, only the first 6 characters of the Advertiser name will be used.


The Spot Insertion file is called XD-SI-YYYYMMDD-XDS Delivery Method.XML (XD-SI-20130709-HBP_Form.xml). If also generating File Delivery, a separate file, XD-FD-20130709, will be generated. 

XD- X-Digital File

SI or FD- Spot Insertion or File Delivery

YYYYMMDD- Date of the Export

X-Digital Delivery Method- is either ISCI, HB, or HBP, depending on your settings. 


If sending the Affiliate Station and Agreement Data, the information exported is: Site ID, Call Letters, Band, Frequency, Owner, Address, City, State, Zip Code, Email, Contact and Phone. 

 

Site ID:  If there is a Station ID defined in the agreement that is what will be sent.  If there is no ID in the Agreement, the ID defined in the Station will be used.  


Address:  The Physical address is sent.  If there is no physical address, the mailing address is used.


Contact:  The contact sent to X-Digital name in station personnel who has the 'Aff E-mail' on.  If no name has Aff-Email, the first name will be used.


Agreement:  The agreement dates must have a date that is the same date as the X-Digital export. Changes to the on air date and last on air date mark the station as needing to be sent. If the receiver (xdReceiverID) changes in the agreement the station will be sent.


Stations:  New and modified stations are sent. (For clients new to Counterpoint, a SQL call can be used to set all stations as new.)
