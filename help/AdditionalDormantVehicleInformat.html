Additional Dormant Vehicle Information

Traffic Reports

Terminated vehicles will not appear in most report dropdown vehicle lists. However, if a user runs a revenue report by vehicle with “all vehicles” selected, and there is revenue for the vehicle in the report period, then the dormant vehicle will be included in the output. If a revenue report is run for a major sort other than by vehicle, and there is revenue from a dormant vehicle in the report period, then it will be included.

 

Revenue Reports and Exports that are affected by these rules are:

  • Billed and Booked 
  • Business Booked by Contract 
  • Invoice Register 
  • Matrix export
  • RAB export
  • Revenue on the Books 
  • Sales Breakout 
  • Sales Comparison 
  • Spot Business Booked 
  • Tableau export


The ReRate report includes dormant vehicles in the output.

Act 1 Research Import

If a research book is imported with data for a dormant vehicle, while the data will import normally, because the dormant vehicle is no longer on the current Rate Card, the data will not go into the Sold Daypart section, as that requires a matching vehicle and daypart on the Rate Card. 

Affiliate System and Dormant Vehicles

Once the vehicle has been set to the Dormant status, the vehicle will no longer be shown on the Affiliate Agreements and Affiliate Affidavits screen when the “Active Vehicle” radio button is selected. To see all vehicles, including dormant vehicles, on the Affiliate Agreements screen, select the “All Veh” (All Vehicles) radio button.

Affidavit Exports and Imports

Even if a vehicle has been set to Dormant, if there are outstanding electronic affidavits that get posted, they will still import normally. This allows affidavits that were generated prior to the termination date to be returned. The vehicle list of the Counterpoint Affidavit Exports screen uses the Export Start Date to filter the list of vehicles. If the Export Start Date is set after the termination date, then the vehicle will be excluded from the vehicle list. If the Export Start Date is set prior to the termination date, then the vehicle will be included, allowing a re-export to be run if necessary (assuming there is an active agreement for that date period).
