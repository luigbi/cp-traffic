Proposals

The Proposal screen is primarily used by account executives to view, create, and edit proposals. When a proposal has been completed and approved, it gets turned into an order or a hold, which will be scheduled, creating spots in the Traffic system. The Proposal screen includes various tools and features to assist with creating and editing proposals, such as the Demo Bar, which shows audience data, and an Avails view, which shows what inventory is available. The Proposal screen is also used to create Proposal XML files for sending to agencies.


In addition to viewing and editing proposals, the Projections feature is also accessed on the Proposal screen. The Projections feature is used by account executives to enter estimates of future contracts, along with the potential of those estimates becoming finalized, as a way to project future business. 


This document contains an explanation of all the features on the Proposal screen and can be used as a reference guide when entering or editing proposals.

Accessing the Proposal Screen

To access the Proposal screen, first launch the Traffic system by double clicking the Traffic shortcut on your desktop, then enter your username and password to login. 


When the Traffic system is launched, you can access the Proposal screen by pressing the Proposals button on the main screen, or by pressing the Proposal button at the top of the screen. 

Basic System Tips

When using a screen such as the Proposal screen, advance from the current field to the next field by pressing the Tab key.


To move back to the previous field, press and hold the Shift key and press Tab while the Shift key is held down.
