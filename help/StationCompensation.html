Station Compensation

The Station Compensation export allows you to create an export file that can be imported into a compensation system to determine the amount due to a station and/or network. 

Setup

Traffic -> Site Options -> Options

In the Options tab, activate Compensation. You will need a keycode from support@counterpoint.net to do so. 

Traffic -> Vehicles -> Options -> Export

Activate the Station Compensation feature on each vehicle whose affiliates are to be included on the export. Checking this on allows the vehicle(s) to be pre-selected in the Export -> Station Compensation screen.

Affiliate Tracking 

Affiliate Agreements

There are three ways in which the system can track compensation due to your affiliates, or due to your network: 


  • Barter: This is used when there is an exchange of station inventory for network content
  • Affiliate: This is used when the station receives monetary compensation for airing network spots
  • Network: This is used when the network receives monetary compensation for network content


The compensation method defaults to Barter, unless a method had previously been defined. If defined as Barter, once spots are posted against the agreement, the Affiliate and Network options will be grayed out. As with any changes to an agreement, you must terminate the existing agreement and create a new one to change the compensation method. If however the method is set to Affiliate or Network, the method can be changed to Barter, but once it has been changed to Barter and posted against, the agreement will have to been terminated and reinstated to make additional changes. 

Stations

The Station ID from the Stations Screen will be sent in the export to identify the station.

Running the Export

From Affiliate ->  File -> Export, select Station Compensation. 


Fill in the export selectivity screen:

  • Dates: The From date must be a Monday and the To date must be a Sunday
  • Use: Select whether to base the compensation/compliance on Feed Dates or Air Dates
  • Compensation Type: Select one or a combination of compensation types to generate: Barter, Pay Station, and/or Pay Network
  • Include Bonus Spots: Not implemented
  • All Vehicles: Select the vehicle(s) to include in the export
    • Vehicles with “Station compensation” checked on in Traffic -> Lists -> Vehicles -> Options -> Export will automatically be selected in the vehicle list
  • All Stations: Select the Stations to include in the export

Export Rules

  • What is included
    • Aired date/time information
    • The export is generated for 12m-12m automatically on all days
    • All Avail Names 
    • Spots marked as Not Aired 
  • What is excluded
    • Affidavits Not Reported 
    • Spots marked as Not carried 

Sample Export

The export file name structure is Station-Comp-mmddyy-mmddyy-A.CSV, and is stored in the CSI\Prod\Export folder.

  • Station-Comp is a constant
  • The dates are the start and end dates of the selected period
  • The letter starts with A and continues on (B, C, D, etc.) if run multiple times for the same period


The file header contains the field names, the second line of the file lists the date and time the export was generated, and the data is displayed in the third file line.
