Rep Reports

The following reports are listed because they are specifically used when working with Rep Orders.  All revenue reports such as Billed & Booked, Sales Commissions, and A/R reports include Rep or have it as an option to include the revenue.

Barter Payments Report

This report uses the terms defined for rep vehicles on the vehicle options screen to calculate the amount due each station per month. (The Barter Payments Export can also be run.)

Delinquent Affidavits Report

This report is used to contact the affiliate for outstanding affidavits for the month selected.

Insertion Orders

The Insertion Order separates the entered order by Vehicle.  This allows each affiliate to receive their information on what to air.  

  • Line Comments print on insertion orders.  Since the Line Comments are associated to a specific vehicle, you donâ€™t have to worry about the comment going to all affiliates.
  • A standard insertion comment that you define in the Site Options will print on each insertion order.
  • Rates can be suppressed
  • Rates can show net-net spot value

Unbillable Invoices Report

This report will show how much revenue is still outstanding that has not been invoiced typically because the posting is not complete.

Rep Invoice Worksheet 

This is used to review the ordered vs. posted spot counts in detail for each station or summary by contract

Salesperson Commission Report

The Salesperson Commission Report will reflect the invoices and the invoice adjustments that are created for Gen Rep and Print Rep Invoice.
