Manually Posting Affidavits

There are three ways of posting manually posted affidavits: CP Receipt Only, Exact Times by Date, and Exact Times by Advertiser. The method used in configured on the Affiliate Agreements screen. 

CP Receipt Only

With this method, click the affidavit week to toggle it between the three available posting statuses: Received, Did Not Air or Outstanding. (The individual spots cannot be viewed with this method. Only the posting status can be updated.)

Exact Times by Date

With this method, the affidavit can be posted with exact times by week. To post the affidavit using this method:


  • Click once on the week you wish to view.
  • On versions prior to version 7.1, a message appears that reads “All Spots Ran as Pledged”. There are four buttons on this screen: Yes, No, None Aired, and Cancel.
    • Yes: all the spots will be posted automatically using the pledged times, and the week will turn to green/Received.
    • No: pressing no will bring up the individual spots for the week where they can be edited as needed then posted. If this option is selected, after posting the times on the following screen, set the posting status to Completed then press Done or Save.
    • None Aired: pressing None Aired will mark all the spots as not aired and the week will be marked as blue/Did Not Air.
    • Cancel: close the pop up message without changing anything.
  • On version 7.1, the popup menu will appear that allows the user to choose between viewing and posting the spots (as described in the prior section).

Exact Times by Advertiser

With this method, the affidavits are grouped by month and year with the spots grouped by advertiser.



To post the affidavit using this method, follow the procedure above for posting by Exact Times by Date as the actual posting process is the same.
