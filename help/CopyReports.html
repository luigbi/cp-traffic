Copy Reports

There are a number of reports that can be used with the Copy screen.

Traffic System Reports

  • Contracts Missing Copy: Lists contracts missing copy for the selected vehicles and dates. This report should be run prior to generating the log to check for any spots missing copy. This report can also show whether copy is “unassigned” – meaning rotations exist but have not been assigned yet, or whether copy needs to be “reassigned” – meaning copy was assigned but was superseded by a later rotation that has not yet been assigned.
  • Copy Book: Prints Live Copy scripts.
  • Copy Inventory Reports: There are a variety of Inventory Reports that can be run to see what inventory exists for various parameters.
  • Copy Play List Reports: The Copy Play List reports list copy airing within the specified date span along with the number of times the copy airs. These reports can be run by Advertiser, Vehicle, or by ISCI.
  • Copy Regions: Lists the stations that make up the selected copy region.
  • Copy Rotations by Advertiser: Lists copy rotations that are in the system for the selected advertiser.
  • Copy Status by Advertiser/Copy Status by Date: Lists spots by advertiser or by date and the copy associated with the spots.
  • Missing ISCI Codes: Lists spots without ISCI codes for the selected vehicles and dates.
  • Regional Copy Assignment/Regional Copy Tracing: Lists generic and regional copy assigned to spots on the selected vehicles for the selected contracts.
  • Split Copy/Blackout Rotation Report: Shows Split Copy and Blackout rotations (no generic) for the selected advertiser and vehicles.

Affiliate System Reports

  • Regional Affiliate Copy Assignment: Lists all regional copy that is scheduled to air or that aired for the selected date range, sorted by vehicle or station. 
  • Regional Affiliate Copy Tracing: Comprehensively lists all regional (and generic if selected) copy airing during the specified date range. This report is meant to be used as an internal tool to determine what regional copy is to air, and to troubleshoot if needed why a particular piece of regional copy did not air.
