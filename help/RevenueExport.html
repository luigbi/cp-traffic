Revenue Export

The Revenue Export creates a CSV file of the data that is generated by the Billed and Booked Report.  The file contents can be imported and sorted various ways, for reporting needs beyond the Billed and Booked options.


The revenue for each contract shows the product, advertiser, agency, vehicle, the primary salesperson, sales source, sales office, participant, cash/trade, airtime/NTR/Hard Cost, political, contract type, and if selected, the vehicle group.  The gross, net and triple net figures are shown for twelve months from the selected start month and year.

Data Rules

The data is gathered the same way it is gathered by the Billed and Booked Report.  The Export does not round the dollars.  

  • Standard Broadcast Month: Receivables/History will be obtained for any dates equal/prior to the last standard billed date defined in Site
  • Projected revenue will be obtained from contracts
  • Corporate Month:  All data will be obtained from contracts
  • Adjustments from Receivables/History are included thru the last billed date
  • All contract types except PSA and Promos are included
  • Trade is included and split based on the percentages defined on the contract
  • Contracts include scheduled/unscheduled holds/orders; no proposals
  • NTR and Hard Cost are included
  • Merchandising and promotions are included for triple net (net-net) figures
  • The spot file is examined for makegoods to adjust for months for missed/makegood spots; rates are subtracted from the missed date, and added on the makegood date
  • Political/non-political are included
  • Installment revenue is used when applicable

Setup

Revenue Export is an option in the Site Options -> Options tab.  Once “Revenue Export” is activated, you must exit and reenter Counterpoint to see the Revenue in the drop down export options.

Running the Export

To run the Revenue Export, go to Task -> Exports -> Revenue, and fill in the necessary information.


  • Start Month: Enter a start month.
  • Year: Enter a start year.
  • Corporate or Standard: Standard broadcast month is the default, unless a corporate calendar is defined in Site, then Corporate will be the default. The export however can still be generated for the standard broadcast year.
  • Contract Number: A single contract  can be selected for verification purposes; if left blank all contracts will be exported.
  • Vehicle Group: Once exported, the data can be sorted by a vehicle group. 
  • Press Export to process. 

File Information

The file is a Comma delimited file with Alpha fields are enclosed in double quotes


Filename:  “Revenue mmmyyyy.csv” i.e. Revenuedec2015 

Mmm = three letter month abbreviation

Yyyy= the year


The file is stored in the CSI\Prod\Export folder
