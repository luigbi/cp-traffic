Audio ISCI Title

The Audio ISCI Title export creates a list of copy for the selected vehicles for the entered date range. The fields shown are: Type, Advertiser, Product, Cart Number, ISCI, Creative Title, Latest Rotation Date, Regional Name, and File Name. Rotations must be created for the advertiser/contract prior to generating this export. 

Running the Export

Click on Tasks -> Exports- > Audio ISCI Title to access the export screen.



Rotation Entered Start Date and End Date: To run the export by the rotation entered dates, enter the start and end date to use in these fields. Any rotations that were entered or edited within the entered date range and that meet the other selected criteria will be included. 


Log Start Date and End Date: To run the export by log dates, enter the date range to use here. 


Export Regional Copy, Export Generic Copy: Choose whether to export regional copy only, generic copy only, or both regional and generic copy by checking the appropriate checkboxes.


Vehicle List: Select the vehicles to include in the export.


Press the Export button to create the export file.


The export file will be saved by default in the CSI\Prod\Export folder. (Alternatively, a non-default path can be used chosen using the Browse button prior to running the export.) By default, the filename is ISCICrossReference.csv. A log file will also be created in the Messages folder with the filename of ExptISCICrossReference.txt.

X-Digital Avail Form Settings

When both the "By Break" and "ISCI" X-Digital avail forms are enabled in Traffic Site Options, radio buttons will appear on the Audio ISCI Title export screen that allows the user to select whether the format of the File Name (on the export output) should use the By Break format or the ISCI format (which are different from each other). This makes it possible for systems that have both avail forms enabled to get the File Name in the correct format for both avail forms. 


Select the "break" format radio button when running the export for "By Break" avail form vehicles, and select the "ISCI" format radio button for ISCI avail form vehicles. Note: if only the "By Break" or "ISCI" avail form is used (not both), the File Name values will be in the correct format automatically and these two radio buttons will not be present on the export screen.

These substituted characters are used:

  • When exporting Short Titles and ISCIs for commercials from Counterpoint to any downstream system.
  • When exporting from a storage system audio files whose file names are made up of concatenated fields such as Short Title and ISCI, e.g., Cingular_Wire(WFSR-4858).mp2. 
  • When posting files to ABC Spot Depot or to other web sites, or to the Starguide, KenCast or X-Digital store-and-forward head ends.
  • When generating line items in a command file for store-forward delivery, such as an envelope or a spot insertion schedule.
