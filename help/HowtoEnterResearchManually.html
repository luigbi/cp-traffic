How to Enter Research Manually

The data on the Research screen can also be manually entered and edited directly on the screen.


To create a new Research Book from scratch:

  1. Select “[New with Demo 18-20 + 21-24]” to create a research book with 18 demo categories where there are separate categories for 18-20 and 21-24, or select “[New with Demo 18-24]” to create a research book with 16 demo categories, with 18-24 as a single category.
  2. Press Tab. Choose to either model from an existing research book or select “None” to not model from an existing book.
  3. Enter the Book Name in the Name field.
  4. Enter the Book Date in the Book Date field.
  5. Enter the Population values for each demo category in the header area of the screen. Alternatively, to use an existing Population or Qualitative Population, select the book to pull from using the “Population Source” or “Qualitative Population Source” field.
  6. Select the method that will be used to store the research data: Sold Daypart, Extra Daypart, Time, or Vehicle. (Multiple methods can be used with a single research book.)
  7. Enter the audience values for each vehicle in the lower part of the screen. (Note: the Group Number is only used for the Radar research import.)
    1. When using “Sold Daypart”, enter the Vehicle Name and Sold Daypart for each set of audience values. The sold daypart should correspond to a daypart entered on the Rate Card.
    2. When using “Extra Daypart”, enter the Vehicle Name, Time, and valid days for each set of audience values. Typically this section is only used when importing research data, as it stores imported data that included a valid daypart but it did not match a daypart on the current rate card.
    3. When using “Time”, enter the Vehicle Name, Time, and valid days for each set of audience values. The time value is used for quarter hour audience numbers.
    4. When using “Vehicle”, enter the Vehicle Name and Days for each set of audience values. Vehicle data applies to the vehicle as a whole, with no specific daypart specified.
  8. Press Save to save the entered information.


Existing data can be edited in a similar fashion by simply typing on the screen then pressing Save to save the changes.
