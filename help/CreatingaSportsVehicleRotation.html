Creating a Sports Vehicle Rotation

A section of the rotation header area changes when creating a rotation in which a single sports vehicle is selected from the vehicle dropdown. When creating a rotation for a single sports vehicle, there are three ways of entering copy: by Time/Day, by Event Number, and by Team. In the rotation header, a special popup menu will appear when accessing the “Time: Start & End” field, as shown in the screenshot below.



These three methods are available for generic rotations, regional rotations, and rotations created using the Copy Grid.

Entering Event Copy by Time/Day

To choose to enter event copy by Time/Day, press the “Time/Day” radio button, enter the start and end time to use, and check on any of the seven days of the week for which the copy being entered is valid.

Entering Event Copy by Event Number

To enter copy by Event Number, press the “Event #” radio button. A list of valid events for the entered rotation start and end dates will appear in the list box, with the event number, event date and team names. The events can be sorted by date or by event number by pressing the “Date” or “Number” radio button. 


To select an event, click it so that it is highlighted in blue. To select multiple events, press Ctrl-Click (hold the Ctrl key down and mouse click at the same time) on the additional events. The rotation that is being created will apply to any events that are selected on this popup menu.



To de-select events, click a selected event to de-select all events or Ctrl-Click on a selected event to de-select only that one event.

Entering Event Copy by Team

To enter copy by Team, press the Team radio button then click the team or teams that the copy is applicable to (the team abbreviations are shown). To select a single team, click on it so that it is highlighted in blue. To select multiple teams, use Ctrl-Click. The rotation that is being created will apply to any teams that were selected on this popup menu.



After selecting the Time/Day, Event Numbers or Teams, tab to the next field to continue entering the copy rotation header and the rest of the copy information following the standard procedure.
