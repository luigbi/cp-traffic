Salespeople

The Salespeople Screen is used to set up salespeople (account executives).


If the commission rate is the same for all contracts and vehicles, you can enter the base and over goal (if applicable) commission rates for remnant (if applicable), and new and increased business commission rates.


  • New Client Business- If the client generated revenue two years ago, but not last year, it is considered a new client.  It will remain a new client all through the current year, and all orders placed in the current year will be given the extra commission.  
  • Rules for calculating Bonus Commission for new business
    • All orders earning revenue in the selected period will be awarded the Base Commission amount, by multiplying the Current Month Net-Net times the Base Commission % stored in the Salesperson screen for each A/E.
    • If the “Last Year” column for the advertiser is blank, the Bonus Commission will also be awarded, by multiplying the Current Month Net-Net times the New Client Bonus Commission % stored in the Salesperson screen for each A/E.
  • Increased Client Business- Increased business is determined by comparing an advertiser’s business this year to the last standard broadcast year (only the last year is compared).
    • Rules for calculating Bonus Commission for increased business
  • All orders earning revenue in the selected period will be awarded the Base Commission amount, by multiplying the Current Month Net-Net times the Base Commission % stored in the Salesperson screen for each A/E.
  • If “YTD This Month” is less than “Last Year,” there is no bonus commission.
  • If “YTD This Month” is greater than “Last Year”, the system subtracts “Last Year” from “YTD This Month” to obtain the commissionable amount, then multiplies the commissionable amount by the Increased Client Bonus Commission.
  • A/E Commissions by A/E, Sub-Company, Contract: this setting in the Site Options ->  Sales tab determines how commission will be entered This determines how Salespeople’s commissions are calculated
    • A/E: If commission is calculated by A/E, the commissions will be based on the rates entered in each salesperson’s profile.
    • Sub-company: This bases a salesperson’s commission on the Jobs -> Commission screen table.
    • Contract: Commission % is entered on a contract by contract basis.


The Job Title field dictates what a user will be able to do throughout the system.

Salesperson
  • Salespeople can only select the Proposal/Order Reports defined with them as the primary salesperson
  • Salespeople do not have the ability to select printable orders
  • The Billing button to allocate promotion or merchandising on the Proposal Screen is disallowed
  • The new Advertiser/Agency screen is modified to require the minimum answer and the accounting questions are skipped
  • When using Projections, they can only enter or modify their own projections
  • The option to use or modify Reservation orders is not shown

Negotiator/Planner

Can do all things that salespeople can do, plus extra capabilities available only to Planners

  • Merchandising
  • Promotion revenue
  • View all Projections
  • Set Projection Blocks
  • Run Projection Roll Overs

Manager

Can do all things that salespeople and Negotiators/Planners can do plus extra capabilities 

  • Approving completed proposals
  • View certain reports
