Invoice Troubleshooting

Unbilled Spots

If you get the red screen that shows unbilled spots:

  1. Check the Message directory for Invoicing.  Select Accessories, Messages, and scroll down till you get to the Invoicing message.  Any error messages that were generated will be displayed.  Email this information to Counterpoint Support.
  2. Print the Log Posting Status for the standard month for unbilled spots only. 
  3. Call Counterpoint Support if you have two invoice numbers for one billing type (Air Time, NTR, or Rep) for one contract.

Interrupted Invoicing

  1. If your invoicing gets interrupted for any reason you should follow the following steps:
  2. Request an Invoice Register by Invoice Number.  This will show what the last contract to be invoiced and the invoice number.
  3. Copy in your ‘zero’ invoicing file.  Select Option Task.  Select Option Reset Work Files.  Call Counterpoint and request assistance if the IVR.BTR file is not copied.  This is the file that stores the invoice data and when invoicing is interrupted, this file is not properly cleared out.
  4. Begin final invoices again.
  5. After they are complete request another Invoice Register by Invoice Number.  Look for the invoice number that was produced at the end of the first report.  Check the report to see for the contract, if there are two invoice numbers or one.  Verify the amount billed for the contract is the correct amount.

Missing Copy

If the copy is missing from the invoice, it could be due to one of the following reasons:

  1. The copy is in the system, but the advertiser or agency was set up to suppress showing the copy on the invoice. Go to the advertiser and agency and see how the question, “ISCI on Invoice” is defined. Change the No to be Yes.  Note:  The Yes/No default for this setting is in Site Options -> Agencies/Advertisers, “Using ISCI Code on Invoices: Always, Never, Ask-default Yes or Ask-default No”
  2. The copy was never defined for the vehicle/contract.  You can look in the Copy Screen at the rotations for the advertiser (or run a Copy Rotations by Advertiser Report). To correct the invoice, you can enter and save the rotation. Use the Assign Button on the Copy Screen to assign the copy to the spots.
  3. If the spots missing copy are ‘fill’ spots, then the question, “Show fill spots on Invoices” may have been set to No instead of Yes. Call Counterpoint Software technical support to get assistance in correcting these spots. 

Erroneous Characters in Agency Address

If odd characters appear in the agency address field, check the following:

  1. Check physical and mailings addresses in the Advertiser/Agency Screens.
  2. Check address for Agency/Advertiser Buyer.
