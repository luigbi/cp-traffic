Sports Rate Card

The Rate Card screen is used to define dayparts for vehicles, set rates (if using the Proposal System), and create Standard Package which can be ordered on all proposals/orders.

Defining Sport Dayparts

Every vehicle must have a daypart defined in Rate Card to be sold. A vehicle can have broad, generic dayparts, such as Mon-Sun 12m-12m, or specific dayparts that ensure specific placement within an event, such as In-Game. Typically, the rate card is defined after the programming has been entered.  A daypart must be created for each unique item, or unit of time, to be sold. 


In this example NBN has one generic daypart of M-Su 12M-12M, while NCAA Football’s dayparts reflect the way spots can be ordered and placed within the broadcast.

Creating Dayparts

When entering (or altering) a vehicle in Rate Card, you can either choose a daypart from an existing list, or create a new one. 

Choosing a Daypart from an Existing List

Any daypart entered in Rate Card can be used with any vehicle. To select an existing daypart for a new vehicle, or to add an existing daypart to a vehicle with dayparts already defined, go to the Rate Card screen and scroll down to the first blank line. From the Vehicle dropdown select the vehicle to which you wish to add a daypart and tab. This will put you in the Daypart field, where you can select the desired daypart from the dropdown.


After selecting the daypart, tab and answer Yes/No to Base Daypart (Base) column.  Base Dayparts are included on the proposal impact button.  This feature will not count any potential revenue booked in a non-base daypart.


Next, decide if you want this daypart to be included on reports such as the Avails report, by answering Yes/No in the Report Daypart (Rpt) field.  

Creating a New Daypart

If you need to create a new daypart, scroll to the first blank line, select the vehicle, and tab into the Daypart field. Select New and Tab, this will bring up the Daypart screen, as shown in the picture below.


On the Daypart screen, enter a name for your daypart. 


Tab through each field, answering all questions. The Base and Report Daypart fields are the same as those described above.

 

Leave Avail Name set to All Avails if this dayparts can book spots into any type of avail. 


Leave Override Hours set to Yes if Overriding the days/times of this daypart is allowed. 


Override Print Rule determines what daypart information will appear on the contract/proposal when a daypart is overridden in order/proposal screen. 

  • If set to None, the Override  times/days will print on the contract/proposal
  • If set to Daypart Name, the Override times/days will print on the contract/proposal with the Daypart name beneath them in parenthesis
  • If set to Avail Name, the Override times/days will print on the contract/proposal with the Book Into Avail name beneath them in parenthesis


Each daypart should have a unique Sort Code number which determines the order in which the dayparts sort on the Avails Report. If left blank, they will sort alphabetically by vehicle.


The Active/Dormant field is used to specify whether the daypart is active and can be selected, or is dormant, and cannot be selected.


Once you have tabbed through all of the header questions, enter 12m-12m as the start time and end time, and check on Monday through Sunday for the valid days. Unlike other vehicle types, sports vehicles must always have dayparts defined as Monday-Sunday, 12m-12m, for 100% of the spots, as the spots are ordered and booked with game events, rather than by date and time.


When finished entering all of the information, click Save and Done, and you return to the main Rate Card screen. Tab through the remaining fields to save the new daypart to the vehicle. 

Creating a ‘Book Into’ Daypart

If you want to control the type of avails spots can schedule into within a daypart, you can use the Book Into daypart feature. This feature allows you to sell a named avail, and those spots ordered with a Book Into daypart will only schedule into that named avail. Using Book Into dayparts will ensure that a spot will air within a specified position in the event. 


To create a Book Into daypart follow the steps to create a new daypart, detailed above. After entering the name of the new daypart, tab into the Avail Name field. Click on the “Avail Name”  toggle 

until it shows “Book Into”. Press Tab, and a list of Avail Names previously defined in the Programming screen will appear. Select the avail that you wish this daypart to exclusively book into.


Follow the steps listed above to complete the creation of the new daypart.

Creating an Exclude Daypart

Similar to the Book Into daypart, which will only place a spot in a designated avail, there may be times when you need a spot to book anywhere except within a certain avail type, for example, a spot can air anywhere in the sports broadcast except in the Post Game show. To handle these situations we have an Exclude daypart feature which is the opposite of the Book Into feature.


To set up an Exclude daypart, follow the steps detailed above to create a new daypart. Once you have defined the new daypart’s name in the Daypart screen, tab to the Avail Name field and click on the “Avail Name” toggle until it changes to “Exclude”. Press Tab and a list of avail names predefined in the Programming screen will appear. Select the Avail Name you do not want the spots to book into, and continue to fill out the rest of the information. 

Creating a Standard Sports Package

If you find that you frequently bundle sports properties with other sports or non-sports properties on proposals/orders, you may want to create a Standard Package in Rate Card which can be selected on all proposals/orders. 


To do so, click on “Std Pkg” along the bottom of the screen. This will bring up the Standard Package screen where you can choose the contents of your package. 


Fill in the header information and then select each vehicle/daypart you wish the package to include by entering a number of spots in the package in the #Spots column. 

Once you have built your package, click Save and Done, and you will return to the main Rate Card screen. The last step in defining a Standard Package is to assign a daypart to the package. Scroll to the first blank line in Rate Card and select the Standard Package name you just created. Tab, and either select an existing daypart from the drop down, or create a new daypart for the package. Fill in the rest of the rest of the fields appropriately, and click Save.

Entering Rates in Rate Card

If you are using the Proposal System, you may enter rates into Rate Card for each daypart sold. Depending on your system set up, Sales may or may not be able to alter these rates while entering proposals.


Rates can be entered by standard Quarter, Monthly, Weekly, or Flight (you determine the number of weeks).


If you choose to enter your rates by Flight, you determine the amount of weeks in each Flight. Click on the Flight button along the bottom of the screen, then click on the #Weeks column header, and the weeks screen will appear. Enter the number of weeks per flight.


Once the entry method is determined, enter each rate against each appropriate daypart.
