Engineering Feed

The Engineering export feeds spot information into a proprietary engineering system. 

Set Up

Traffic Site Options

You must have “Using Delivery Vehicles” checked in Traffic Site Options to use the Engineering Export. Activate the links in the Site Options -> General tab.


You must have the “Pre-Feed” option checked on in Site Options -> Options tab, under System Options.


If you are using the ASP (Counterpoint Engineering) mode, and you can suppress the Media Code(s) from being included in the export by checking “Engineering/Cart Export – Suppress Media Code Except “L” (Live)”, in the Site Options -> Export tab. Any cart with a Media Code of L for Live, will still be included in the export. 

The first line is using a non-L media prefix, the second line is using a cart with the L media prefix):

151215 00 214500 12157 060 USPS,STAMPS 0030 USPS-123 

151215 00 214530 12157 L007 MCDONAL,BIGMAC 0030 M123

Vehicle -> Options -> Export

Any vehicle that will be included in the Engineering export must be set up in the Vehicles -> Options -> Export screen as using the ASP or ESPN export mode. 


If a sports vehicle is to be included in the Engineering export, checking “Engineering Feed – ASP”, will allow you to define it’s Bus in the Event Schedule rather than in Engineering Links.

Avail Names

Spots from any avail name with the Automation Export setting set to No will not be included in the Engineering Export.  

Pre-Feeds

Delivery Link Pre-Feeds can transmit programming/spots to the affiliate prior to their actual sold time/dates, to be recorded for playback at another time/date.


The Pre-Feed screen will show the date and time that the vehicle will be Pre-Fed, or transmitted in advance. 

Pre-Feed Rules

  • Vehicles must be using Delivery Links to access the Pre Feed input.
  • Logs are not produced for Pre Feeds.
  • If Pre-Feeds are defined for Delivery Links, they are disallowed for Engineering Links.
  • If Pre-Feeds are defined for Engineering Links, they are disallowed for Delivery Links.

Running the Export

To run the export, click Task -> Export -> Engineering, and select the type of export you wish to run. ASP is the Counterpoint Engineering system. ESPN is a client specific feed. 


Export Date: Enter the export date


Vehicles: Select the vehicles to export.


Blackout: If necessary, create any blackouts and replacement spots.


Click Export to run the export.

File Information

The file will be stored in CSI\Prod\Export.


Each export type has its own file naming structure:


ASP: yymmddAE.Txt


ESPN: yymmddESPN.Txt


For both files, yymmdd is the year, month and day of the date being exported.


The Export-Engineering Feed log file, viewed in Accessories -> Messages, will list everything exported, as well as any errors that occurred during the process.
