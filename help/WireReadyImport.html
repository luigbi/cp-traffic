WireReady Import

Fixed Length, 127 bytes, Filename: YYMMDDSSSSS.rec.  All records terminated with c/r l/f.

(YY = year, MM = month, DD = day, SSSSS = unique 5-character station code that must be entered into the Station code on the Vehicle screen of the vehicle being generated).


Same format as what is sent to WireReady, except with last 3 fields added.  Filename is the same, except .REC is used for file extension instead of .TXT.
