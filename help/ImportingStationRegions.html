Importing Station Regions

You can import a group of stations to be included in a region from a .TXT (text) file. 

Creating the Import File

The station import file can be created using any text editor program such as Wordpad, Notepad, etc. The extension of the import file must be .txt and each line within the file must only contain the call letters and band, and end with a carriage return, as shown in the example. 


Save the import file in the CSI\Prod\Import folder.

Importing the Station File

From the Region screen, press the Station tab. The “Import” button will become active when on the Station tab. Click  “Import”  and select the text file from the File Name area on the left side of the screen. The stations on the file will appear in the box beneath the File Name area. 



Click “OK” to import. 


The stations on the import file will be highlighted on the Station tab, and can be moved into the Results field for inclusion on the Region by clicking “Include”.      



Any station in the import file that is not found in the Station list will be recorded in the file “StationsNotFound.txt” and stored in the CSI\Prod\Data\Messages folder.


The contents can be viewed from the Accessories item Messages Viewer.


Once the stations have been added, press Save to save the new region.
