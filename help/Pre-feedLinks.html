Pre-feed Links

Pre-Feed date/times are defined to allow programming/spots to be transmitted to the affiliate prior to their actual sold time/dates, and recorded for playback at another time/date.


The Pre-Feed screen will show the date and time that the vehicle will be Pre-Fed, or transmitted in advance. 


The exports that will use Pre-Feed are: 

  • Engineering Export- EAS 
    • Uses Engineering Pre-Feeds
  • Engineering Export – ASP 
    • Uses Engineering Pre-Feeds
  • Dallas Export
    • Uses Delivery Pre-Feeds 
  • X-Digital

Pre-Feed Rules

  • Vehicles must be using Delivery or Engineering Links to access the Pre Feed input
  • Logs are not produced for Pre Feeds 
  • If Pre-Feeds are defined for Delivery Links, they are disallowed for Engineering Links
  • If Pre-Feeds are defined for Engineering Links, they are disallowed for Delivery Links


X-Digital Pre-Feed Rules

The Pre-Feed feature will allow spots to be transmitted via XDS prior to their airing date/time. 

 

The XDS system can transmit spot information three ways:

  1. Record the program with split copy
  2. Record the program without split copy and play it back at a specified time within the X-Digital equipment 
    1. This is the option that requires the X-Digital export to be sent each day between the Pre-Feed day and Sunday.
  3. Play the program live


For Pre-Feeds to work with XDS, the vehicle must air on Saturday OR Sunday, one day only, the network must be using the ISCI Avail XDS method, and be utilizing Engineering or Delivery Links. If the vehicle airs on more than one day over the weekend, this method will not work.


The X-Digital export can be generated for any day between the pre-feed day and through Sunday, and the system will automatically adjust the dates, even though the vehicle only airs on one day. This is because Counterpoint doesn’t know when the affiliate is actually going to air the program. 


Example: If a vehicle normally airs on Sunday 4/6, and there’s an AC Delco spot airing, and the pre-feed day is Thursday. The X-Digital export can be generated for Thursday 4/3, through Sunday 4/6, and the spots on the report will be the same, with just the date changed. So that AC Delco spot will appear on the Thursday, Friday, Saturday, and Sunday export.

Setting up Pre-Feeds

Turn on Pre-feeds in Site Options -> Options. You will need a keycode from support@counterpoint.net to do so.


Delivery Pre-Feed Links

Delivery Link Vehicles 

  1. Select Link on the programming Screen
  2. Select the day the links are defined on 
  3. Select Delivery button
  4. Select "Define Delivery Links" 


From the Links button, enter a start date that matches the start date of the Delivery or Engineering links that have already been defined. Click either Define Delivery or Engineering Links. Select the Pre-Feed button, to bring up the Pre-Feed entry screen where you define the feeds.



1. Enter the "From" start and end time, day, zone

2. Enter the "To" start time and day


Note:

  • All times are local affiliate times
  • The "To" day must be prior or equal to the "From" day
  • Multiple Pre-Feed Times/Days may be defined


Press Save to save changes or Cancel to exit without making changes.


Engineering Links

The steps are identical for entering Engineering Pre-Feeds, but with one additional step: you must define the Bus information.


In the Pre-Feed screen, select the Pre-Feed Line and enter the bus pre-feed from/to info for that line.
