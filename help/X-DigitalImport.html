X-Digital Import

Airing Dates and Times and copy information can be imported directly from X-Digital and posted automatically on the Counterpoint Affidavit website, so that station users do not have to manually post the airing information for those spots.


The X-Digital import runs directly on the Counterpoint Affidavit website, automatically, and on a schedule. It is not run manually by users and does not appear in the list of affiliate imports on the affiliate import menu.


When it runs, it checks for airing information from X-Digital and imports aired dates and times when they are ready to be imported into the web spot database. When airing information has been imported in this way, station website users will see the imported airing information for the spots on the electronic affidavit screen. Spots that have been posted by the X-Digital export will appear as posted on the affidavit screen in the “submitted” area at the bottom of the screen. The spots that were not posted by the X-Digital import will need to be posted manually like normal. (Once all the spots have been fully posted, the affidavit will become available for importing back to the affiliate system as before.)


The X-Digital import reduces the number of spots that need to be posted manually and helps reduce the possibility of data entry errors because the airing information is imported directly from X-Digital.

Configuration Information

The X-Digital import is supported by all avail forms (the two “by break” avail forms: Hour/Break, and Hour/Break/Position, and the “ISCI” avail form), when the “Unit ID by Affiliate Spot ID” option in Traffic Site Options -> Export is checked on for the corresponding avail form. 


The Vendor Setup screen must also be configured correctly, following these instructions:


  1. On the Affiliate system, go to File -> Delivery Vendors and enter the password supplied by Counterpoint support.
  2. Select the “X-Digital-Break” option from the dropdown menu on the right (for the By Break avail form), or “X-Digital-ISCI” for the ISCI avail form.
  3. Enter the Host address provided by X-Digital in the “address” field. (This is the same “host” value that is entered on the XML.ini in the X-Digital section.)
  4. Enter CSI as the “Sign On Name”.
  5. Enter the security token provided by X-Digital in the “Password” field. (This is the same “Security Token” value that is entered on the XML.ini in the X-Digital section.)
  6. Press Save and then Done.


The import must also be configured on the Counterpoint Electronic Affidavit website. Please contact Counterpoint support for assistance in setting this up.

Import Overview

The following is a general overview of the workflow when using the X-Digital import:

  1. Generate the final log for a vehicle that uses X-Digital.
  2. Run the X-Digital export to send the copy information to X-Digital.
  3. Export to the web using the Counterpoint Affidavit export. 
  4. When the spots air, the airing information will be updated on the X-Digital system.
  5. On a schedule, the import program on the website will be checking X-Digital for posted spot information. When it finds it and it meets the other rules that mean it's ready for importing, the spot info is imported to the website database and the spots are updated on the web.
  6. The station website user posts the remainder of the spots so that the affidavit is now fully posted.
  7. The CSI Electronic Affidavit import is run, importing the spot info from the website like normal, only now it includes the airing info from X-Digital along with the manually posted airing information that was posted by the station user.
