Split Networks

The Split Network feature makes it possible for different advertisers to air different spots in different regions within the same avail. 


For example, a 9am avail could have both advertiser A that airs on LA stations, advertiser B that airs on NY stations, and advertiser C that airs on all the remaining stations. In this example, a contract would be entered for advertiser A using a split network region that includes the LA stations, and another contract would be entered for advertiser B using a split network region that includes the NY stations, and a split fill definition would be entered for advertiser C that will automatically fill in for the split network spots on those stations that are not included in the split network regions that were defined for advertisers A and B.

Split Network Rules

The following rules apply to split networks:


  • Split network spots and split network fills can air on the following vehicle types: conventional, selling/airing, and sports vehicles.
  • Split network regions are indicated on the contract line.
  • Split network regions can be defined by format, market, state, station, and time zone. 
  • Split network spots will air only on the stations included in the specified split network region.
  • Stations that are not included in the split network regions must be filled using the Split Fill feature on the Logs screen (or there will be no spot for that station when viewed on the Affiliate system where the split network spot is on other stations).
  • There is no limit (up to the total number of stations defined in the Affiliate system) to the number of advertisers that may share an avail.
  • Split network spots will appear on the Spots screen with a light cyan (green-blue) background color. The advertiser that has purchased the most stations will be considered the primary spot and will appear above the non-primary split network spots in the avail. The primary spot will also have a slightly darker cyan background color.

Basic Procedure Overview

  1. Enter a split network contract. Select the split network region for the line in the “Region” field of the override box.
  2. Save and schedule the contract.
  3. Enter copy for the split network contract.
  4. Enter any additional split network contracts as needed.
  5. On the Logs screen, press the Split Fill button and enter the split fill information for the alternate advertiser that will air for any station not included in the split network regions covered by the contracts that were entered.
  6. Generate the final log.
  7. The split network spots will air on those stations that are part of the split network region defined on the contract line. The split fill spots will air for those stations that are not include in the split network regions on the contracts.
