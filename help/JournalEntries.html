Journal Entries

Journal Entries are non revenue transactions. They are used to debit or credit an account. Journal entries can only be entered in the current period, and can only be posted against due amounts.  


Journal Entries show on your Cash Receipts Reports. The Accounts Receivable Reconciliation Report lists the totals for each Journal Entry type for your General Ledger totals.


When you apply journal entries, if the invoice has multiple transactions the journal entry will be proportionally applied to the invoice transactions. There are three options that can be used to apply journal entries:


  • Invoice: This groups together all IN transactions for multiple vehicles on an invoice. When you enter an amount to adjust, the system will proportionally apply the adjustment to all vehicles on the invoice
  • Billing Vehicle: This groups together all IN transactions with the same billing vehicle to allow you to adjust a portion of an invoice
  • Airing Vehicle: This lists each transaction individually allowing you to adjust any element of an invoice


The system defaults to the Invoice method. To change the Journal Entry method, click the “JE Inv” toggle to “JE Bill Veh” or “JE Air Veh”. 

Transaction Types

Every Journal Entry must have a two character transaction type code that begins with “W”. You can select the transaction type from the list defined, or enter a new two-character code. 

Creating a Journal Entry Code

To create a new Transaction Type, select New from the Type field dropdown. Enter a description, or name for the Transaction, enter a two letter character code, and click Save. The code must start with a W.

Entering a Journal Entry 

To enter a JE (Journal Entry):

  1. Select the agency and the Journal Entries action
  2. Enter the Transaction Date
  3. To apply an +/- amount to a transaction, mouse click in the Journals column. You must click on a line that has an amount due
  4. If it is a credit amount, you must enter the "-" before the dollar figure
  5. The transaction amount will move to the Journals column
  6. If the amount needs to be changed, mouse click on the amount in the Journals column and enter the correct amount
  7. Select a transaction type to define the Journal Entry, or create a new one
  8. You can attach a comment to a JE by clicking in the C column 
  9. The Due column will automatically increase/decrease by this amount
  10. The Total Due amount will automatically increase/decrease by the amount selected
  11. The Total Journal amount will automatically decrease/increase to the selected amount
  12. Pressing Save will create a transaction line for the JE, with the Transaction code you assigned the item
  13. If a comment was attached, a “C” will appear in the “*” column
