Purging Expired Copy

The Purge screen allows you to purge previously used and expired carts so that they can be reused. 


Press the “Purge” button at the bottom of the Copy screen to access the Purge screen.



The following is an explanation of the different fields on the Purge screen.

  • Media Code: Select the media code for the inventory you want to purge.
  • Rotation End Date: This will bring up all inventory items that are expired as of the entered date. (This date must be earlier than today’s date.)
  • Advertiser Dropdown: Inventory can be purged for a single advertiser by selecting it from the dropdown menu, or for all advertisers by selecting “N/A” from the dropdown menu. 
  • To Be Saved: Carts in the “To Be Saved” list will remain in that state after purging. 
  • To Be Purged: Carts in the “To Be Purged” list will be purged when pressing Done or Save on this screen.
  • Move Buttons: Carts can be moved from the “To Be Saved” list to the “To Be Purged” list or from the “To Be Purged” list to the “To Be Saved” list using the Move buttons.
  • Sort by Rotation End Date, Advertiser, ISCI, Cart #: This will determine the order in which the inventory will be sorted


Depending on the Cart Disposition setting, expired carts will either be listed in the “To Be Saved” list or the “To Be Purged” list. 


Press the Save button to purge any inventory in the “To Be Purged” list.


When entering a new piece of copy, purged cart numbers will be available for assignment. 


On version 8.1 and above, when selecting the advertiser or all advertisers, the "Set Copy Inventory Dates" process is automatically triggered to check and fix any incorrect earliest/latest rotation dates for the selected media code inventory before making the inventory available to be purged as a precautionary step.
