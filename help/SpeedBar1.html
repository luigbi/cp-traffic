Speed Bar

The Speed Bar is a way to quickly and easily add lines to a contract. Both conventional lines and package lines can be created using the Speed Bar.

To create contract lines using the Speed Bar:

1. From the Specifications view, enter the Start Date, which is the date the lines you will be creating should start. (The view can be switched between Specifications, Vehicle Summary, and Daypart Summary.)


2. Enter the number of weeks in the “# Wk” field or enter an end date for the lines in the End Date field.


3. The “Weeks On” and “Weeks Off” fields are used to alternate weeks on and off. To add spots in each week of the entered date span, set the “Weeks On” to 1, and the “Weeks Off” to 0.


4. Enter the number of spots per week in the “Spots/Week” field. 


5. To the right of the End Date, the weeks that will be created appear, as shown in the picture below, along with the spot counts to the right of the week date. You can Ctrl-Click on a week to deselect it so that it won’t be created.

 

6. To the right of the week dates, the available vehicles are listed. Click or Ctrl-Click on vehicles to select them. (Sports vehicles are excluded from this vehicle list.)

7. The dayparts that are on the rate card for the selected vehicles will appear in the daypart area, to the right of the vehicle list. Click and Ctrl-Click dayparts to include them.


8. Click and Ctrl-Click can be used to select the spot lengths for the lines that will be created.


9. To use a specific book for the lines you create, select it from the Book Name field.


10. After filling out the top part of the screen, press the Generate Grid button. The grid will be populated with the vehicles, dayparts, spots, etc., for the lines that will be created.

11. Place a checkmark in the leftmost cell to check a row on that you want to create lines for.


12. The spot counts, price, CPP, and CPM can be edited if needed. The Set Value fields above the Spot Count, Price, CPP, and CPM can be used to edit those values. Press Tab after typing in the new value. Altering the price value will automatically alter the CPP and CPM fields, and altering the CPP or CPM will automatically alter the price and CPP or CPM using the new value. The picture below shows the Set Value fields being used to change the Spot Count to 4 and the price to 5,000.


13. The Spot Multiplier in the lower right corner is used to change the spot count for each row based on the entered multiplier value. For example, if the spot count was 3, and you entered a spot multiplier of 2, the spot count would change to 6 for each row (when pressing Tab).


14. After the rows have been configured, press the Gen Lines button to generate the lines on the contract. Alternatively, if the selected vehicles consist of non-package vehicles, press the Gen Pkg Line button to generate a package (see below for details).

Generating Package Lines with the Speed Bar

When only non-package vehicles are selected on the Speed Bar, you can create a package directly from the Speed Bar. The following options are available:

Package Type: select Real (if allowed) or Virtual, and enter the number of virtual spots per week when using the Virtual option (Real package spot counts are equal to the number of hidden line spots).

The “Package By” section is used to select the remaining package options.

Create Package for Each Daypart: when creating packages with this radio button selected, each different daypart will have a separate package created for it. Enter the package name in the package name field, or select a name from the vehicle list.

When using the “create package for each daypart” option, press “Yes” on the “Place vehicle names into Other Comment” to add the hidden line vehicle names to the “Other Comment” field in the contract header area.


Create Package for Each Vehicle: when creating packages with this radio button selected, a separate package will be created for each vehicle, where the package name is set to the hidden vehicle name. Select the daypart that will be used for the package from the list of dayparts, or enter a new daypart in the text entry box.

Create One Inclusive Package: when creating a package with this option, a single package will be created, using the selected or entered vehicle name for the package name, and the selected or entered daypart.

After selecting the package options, press the Done button to create the package or packages.
