Merging Spots Between Programs

Since X-Digital cannot air multiple programs simultaneously, if spots need to be merged from multiple programs into multiple programs, enter MERGE in the Program ID field and check Honor Program ID Merge. 



The spots on any vehicle with a program ID of MERGE and that has the “Honor Program ID Merge” option checked on in the Vehicle options>Interface tab, will merge onto any vehicle with programming library start and end times that encompass the avail times. Meaning that if a vehicle set to merge airs between 3p-5p, the vehicle(s) onto which the spots will merge must have a Programming library that is at least 3p-5p. If the vehicle(s) onto which spots will merge has a shorter library than the merge vehicle, you must increase the library’s length by extending the program’s library end time so that all Merge program breaks are encompassed within the library. 


The Break numbers will be adjusted so that the Merge spots will appear as if they were always part of the exported program.

Example:

The program that will carry the Merge breaks from another program have the following breaks defined: 6:10a; 6:20a, 6:30a, 6:40a; 6:50a. The Program Library start time is 6am and the end time is 7am (determined from the program length).


The program to be merged has the following breaks: 6:15a; 6:35a and 6:55a
