Additional Post Buy Planning Features

Call Letter Search

If there are too many stations displayed on the screen, you can type in the call letters that you are looking for in the “Call Letter Search” field at the bottom of the screen. If the call letters are found, the screen will jump to those call letters in the list.

Station Email

Emails can be sent directly from the system using the Station Emails button. 

Station Report

Press the printer icon in the bottom right corner to print a snapshot of the station information shown on the screen. This report has four ways it can be run: Contact Information, Vehicle List, Contact and Vehicles, or None.

Contact Information: If selected, all contact information will appear on the report.


Vehicle List: If selected, all of the vehicles that meet the filtered criteria will appear on the report.


Contact and Vehicles: This is the default selection. If selected, the station contact information and the vehicles that meet the filtered criteria will be listed.


None: If selected, no additional information will appear on the report (contact info and vehicle info is excluded).


Note that the heading of the report displays the filter information from the top part of the Post-Buy Planning screen to indicate what criteria was used when generating the report.
