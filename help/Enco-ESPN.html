Vehicle Screen

Selecting ENCO-ESPN in the Vehicle -> Options -> Export Screen will allow vehicles to be automatically selected on the export screen. 


This step must be done for every new vehicle entered into the system, in order for the vehicle to be automatically selected.

Programming Screen

Every Studio Log vehicle and Sports vehicle to be exported must have Program Library Event Names defined in the Programming -> Library screen. This Event Name will appear in the export file.


To define the Name, select Programs from the Event Type field dropdown, and tab into Event Name, and select name from the dropdown.


To create a new Event Name, Select New from the dropdown, press Tab, and enter the new Event Name. 

The Automation File

The export produces a Comma Delimited File that contains the following information: 


     Field Name                                Format          Max Length

    '   Date                                         m/dd/yy                 8 (month, date, year)

    '   Time                                         hh:mm:ss        8 (military time)

    '   Length                                      nnn                         3 (numeric)

    '   Event Name                             AN                  20 (alpha numeric) 

    '   ISCI Code                                AN                  20 (alpha numeric)

    '   Advertiser/Product                   AN                         35 (alpha numeric)

    '   Progam ID Hour Break Position         AN                     (alpha numeric)

    '   Internal Code                                      nnn                10  (numeric)


Including the Hour and Break Position allows adjacent avails to be treated as separate avails in the ENCO system, matching the X-Digital export.

File Naming Structure

For non-sports (conventional vehicles), the file name contains the Vehicle Name and XDS Program ID, followed by the Month, Date, and year. Example: VehicleName_XDSProgramCode_MMDDYY.csv. 


For Sports vehicles, the file name contains the Vehicle Name, the XDS Program ID, and the Visiting v Home Team names. Example: VehicleName_XDSProgramCode_VisitingTeam_Vs_HomeTeam.csv.

File Location

The automation file can be stored in one of two places. The first place the system looks for a file is in an export folder defined in the Vehicle -> Options -> Log screen, in the Automation Export Drive/Path field. Each vehicle can have a unique export folder. 


If a vehicle does not have a unique export folder defined in the Vehicle -> Options -> Log screen, the file will be stored in the export folder defined in the Traffic.INI. Typically, it is stored in CSI\Prod\Export.
