CPTT Check

The CPTT records indicate which week logs have been generated for in the traffic system and sent over to the affiliate system, and retain the Posting status by affiliates.  The CPTT records should always have a Monday date and only one record should exist per week for each agreement.


The purpose of CPTT Check is to add missing CPTT records and to remove extra CPTT records.   


The Utility does the following:

  1. Remove any CPTT that are prior to the start of the agreement
  2. Remove any CPTT that are after the end of the agreement
  3. Add in any missing CPTT within the agreement start and end date
  4. Remove CPTT within agreements that don't belong
  5. Remove duplicate CPTT
  6. Set the posting status

 

To run the Utility, click on File -> Utilities -> CPTT Check, and enter the date range to check.


Click “Check All” or “Check Posted Only” and the results will appear in the white box, along with a message saying the utility completed successfully. 


If errors are found, click Fix All to correct them.
