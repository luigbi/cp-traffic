Duplicating Inventory to Another Media Code

If carts are used and there is more than one media code defined in the system, it is possible to duplicate copy inventory information from one media code to another by following this procedure:


  1. From the Inventory field, double click the piece of inventory that you want to duplicate to bring up the Copy Inventory screen.
  2. Press the Duplicate button.
  3. Select the media code that you want to create the duplicate for from the left side of the screen.
  4. Select the cart number that you want to duplicate the copy to from the right side of the screen then press Save, then Done.


In this image, the media code “D” has been selected to copy the cart information to that media code, and cart 001 has been selected on the right side of the screen to duplicate the cart information to cart 001.


  1. After returning to the Copy screen, the new piece of copy inventory will now be appearing in the Inventory area under the new media code (the original inventory will be unchanged). As part of this process, the Length, ISCI Code, Creative Title, and other applicable values will be copied to the new piece of inventory. (The number of times aired and other airing information does not get copied.)
