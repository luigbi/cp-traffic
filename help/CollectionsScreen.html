Collections Screen

If you have any questions about what any abbreviation or color means throughout the Collections screen, click the Key icon in the upper left hand corner for information about the screen.

Working in the Screen


Different colored fields determine what can and cannot be done:

  • Yellow– This information is automatically updated and cannot be altered
  • Light Blue – Mouse Click to move or select information
  • Dark Blue-  Indicates a toggle
  • White  - Allows data input


Transactions may have notations that will appear in the “*” column.  These notations indicate the action that was taken for the transaction. This information is also located in the Key icon.


R

Reversal Transaction

B

Returned (bounced) check

D

Redeposit Check

I

Transferred In

O

Transferred Out

A

Cash from On Account

C

Comment on transaction


When entering the Collections screen, it is assumed you will want to sort and view information alphabetically by Agency. To change the way information is selected, select a different view from the dropdown in the upper left hand corner. Other options show the clients based on credit restriction, payment rating or amount overdue. This sort is defaulted to alphabetical by agency; this is the only sort that allows posting of transactions.



Select the Agency from the dropdown at the top of the screen near the center. Any outstanding amounts owed will be shown next to the agency’s name.


Revenue Types 

The Collections screen defaults to showing you Cash transaction types, but there are other revenue transactions that can be viewed:

  • Cash - for all current cash receivables
  • Cash History  - for all zero purged transactions
    • Note:  If you use installment billing, revenue transactions may be listed here
  • Trade -  for all current debit trades
  • Trade History -  for all zero purged transactions
  • Merchandising -  for all current debit-merchandising amounts
  • Merchandising History  - for all zero purged transactions
  • Promotion  - for all current debit promotional amounts
  • Promotion History  - for all zero purged transactions


You can select the revenue type to view from the Type dropdown.

Collection Actions

Collection Action determines what you will be doing in the Collections Screen.


  • Ageing: Offers an overview of outstanding amounts
  • Post Cash: Used to apply a check/cash to an invoice or leave on account
  • Apply On Account: Used to apply a Payment On Account (Pre-pay) to an invoice 
  • Transfers: Used to move payments from one invoice to another, or one advertiser to another with the same agency
  • Reversals: Used to back out an entered transaction in the current period
  • Returned Checks: Reverses all cash postings paid by a bounced check
  • Redeposit Checks: Reposts a bounced check when made good, and reapplies the payments
  • Journal Entries: Used to add a non-revenue offsetting transaction
  • Invoices: Used to enter a revenue transaction
  • Invoice Adjustments: Used to add invoice adjustments to a transaction


On version 7.1 and above, for all the different actions (excluding Invoices), the transactions that appear on the screen can be configured to combine all the vehicles for a single invoice into one row, as shown below. 


The row can be expanded so that all the associated vehicles can be seen by mouse-clicking the down arrow to the left of the Name column. 


Clicking the up arrow that appears for the last line will collapse the invoice back to a single line of information, or clicking the arrow next to another invoice to expand that invoice will collapse the expanded invoice back to one line (in other words, only one invoice can be expanded at a time).


To enable this feature, check on the “Compress Transactions” checkbox on the Traffic Site Options screen, on the Accounting tab.

Right Mouse Information

Additional transaction information can be viewed by holding down your right mouse on a transaction in any action screen.



  • When viewing detail invoice information, you will see Invoice or NTR Invoice (indicating if it is an Air Time Invoice or NTR)
  • The Bill Vehicle would be the ‘conventional’ or package vehicle that appeared on the invoice
  • The Air Vehicle is the vehicle where the spots aired and what amount for that vehicle has been updated
  • Cash transactions will display their check number 
  • The Transaction Date is the entered date for any cash or journal entries, the last date of the standard broadcast month for invoices
  • The Type indicates the two-character transaction type for this transaction (IN, AN, etc.)

Collection Comments

Collection comments can be used to give additional information about the transactions that were entered or to make notes about actions that will be taken. Collection Comments print on the “Advertiser or Agency Credit Status Report” when the “Include Comments” checkbox is checked on.  


Collection comments are entered near the lower part of the Collections screen. To enter a comment:


1. Select an agency or advertiser from the top of the screen.

2. Click in the “Action Date” field and select an action date (or the date on which action will occur). The default is seven days from today’s date.

3. From the Advertiser/Agency field, select the agency or advertiser, whichever the comment should apply to.

4. Click in the Comment field and type in the comment text, then press Tab. The entered comment will save automatically, and be displayed in the white area near the bottom of the screen.



Each entered comment will show the Entered Date (the date the comment was entered), the Action date, and the comment text. Use the scroll bar on the right side of the screen to scroll through the entered comments.


Note: Saved comments cannot be edited or deleted.
