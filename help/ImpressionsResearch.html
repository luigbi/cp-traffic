Impressions Research

Impressions Import

The Impressions import is used to import research by impressions for use with podcast spot buys. The import screen is shown below.


Only systems with the Traffic Site Options setting of “Show Impressions and Download on Proposals/Orders” checked on will have access to this research import. (This setting appears on the Site Options screen Audience tab.)

 

The imported data is stored with the demographic category of P12+ (no other demographic categories are used with this import), and a special screen appears on the Research screen for books that have been imported using the Impressions research import.


An example of the contents of the import file is shown below:


Vehicle,Daypart,Impressions,,,,

Podcast A,Mid-Roll,"300,000",,,,

Podcast B,Pre-Roll,"250,000",,,,


The “Vehicle” corresponds to the vehicle name, the “Daypart” corresponds to the daypart/ad location name, as defined on the Rate Card and Daypart screens, and the “Impressions” is the estimated downloads per episode stored with the P12+ demo category.


To import research by impressions:

  1. Browse to the import file (a comma delimited file in TXT or CSV format), and enter a book name and book date in the corresponding fields. 
  2. To set the new research book as the default vehicle book (for those vehicles included on the import file), check the “Rating Book Name” checkbox. 
  3. Press Convert Files to begin the import process.

Impressions Research Screen

When Impressions-based research books are used, a special screen appears for displaying this type of research book, as shown in the picture below.

Research books by impressions only display the Book Name, Book Date, Vehicle Names, Dayparts/Ad Locations, and Impressions. There are no audience values or demographic categories (all impressions are stored with the P12+ demo category). Impressions shown here use the order of magnitude specified in Site Options on the Audience tab under the “Audience Data in Thousands/Hundreds/Tens/Units” setting.


Impressions research books can be imported using the Impressions research import (see the Traffic Imports help document for more information on this import), or manually created (and edited) on the Research screen.


When manually creating a new research book from the Research screen, to create an Impressions based research book, on the Model From screen, check the “Impressions” checkbox (or model from a previously created impressions based book), as shown in the picture below.


The plus/minus audience adjustment can be used to increase or decrease the impressions from the modeled from book by a specified percentage.
