X-Digital Testing Procedures

Traffic Steps

  1. Open Copy Screen
  2. Click the advertiser dropdown and pick the advertiser you wish to enter the split copy for. Make sure it’s for an advertiser that is on a vehicle that is airing on the Test Station being used for this test.
  3. Click on Region at the bottom of the screen.
  4. Fill out a new Region for the Test Station and click Save.
  5. Fill out the two Rotation lines for the Test week.  When copy is assigned it looks at the rotation based on last entry first used. The generic rotation must be entered first.  
  6. First Rotation is for the Generic Copy. Add the Cart # that should go to all stations.
  7. Second Rotation is for the Split Copy. Add the Cart # that should only go to the Test Station/Region you setup.
  8. Once you have all other copy entered into the system and you are ready to close logs you can run final logs (and assign copy). This will push all the spots over to the Affiliate System.

Affiliate Steps

Here are two ways you can easily verify that the split copy got assigned correctly:

Advertiser Fulfillment Report 

  1. Run the report with the setting to show split copy for the Advertiser you entered the Split Copy for ton confirm that the split copy shows up on the Test Station. 
  2. If the Report is showing the Split Copy, then it worked and you can proceed to the next step.  
  3. If the split copy is not appearing, review the setup steps to determine why the split copy is not showing. Double check that the test station matches the region criteria that was defined. Double check that the copy rotation information was entered correctly for the test contract that was entered.

Affiliate Affidavits

  1. Bring up the test station on the Affiliate Affidavits screen.  
  2. Click on the red week you entered the split copy for and click “no” for the question did all spots run as pledged.  The week will open and you should see all the spots for the week.  
  3. Click on the column heading to sort by Advertiser/Product to find the advertiser you did the split copy for.  
  4. Put you cursor over the air date or air time for the test advertiser’s spot and you should see what ISCI Code is assigned to the spot. If you see the ISCI Code for the Split Copy, then it worked and you can proceed to the next step.  
  5. If the generic copy only is appearing, review the setup steps to determine why the split copy is not showing. Double check that the test station matches the region criteria that was defined. Double check that the copy rotation information was entered correctly for the test contract that was entered.

ISCI Cross Reference Export

  1. Give this Report to the person that will name the Audio Files to make sure they name each spot exactly how Counterpoint has the file named (this applies to both Generic and Split Copy spots).
  2. This person should also place these spots in the Spot Depot folder that X-Digital pulls the spots from.

X-Digital Export 

Run two exports: 

  • Generate an XML Test File. This generates a file in the csi\prod\export folder that can be used to confirm that everything was entered correctly.
  • Send to X-Digital. This export is sent to X-Digital.  If you receive no errors then the connection went through and you should see the Split Copy information on the XDS-NMS System.

Final Test

Wait until the spot airs on the test station and see if the generic spot is pulled and replaced with the split copy/region spot.
