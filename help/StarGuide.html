StarGuide

This will produce an export file for the StarGuide System.

Setup

Traffic: Site Options -> Options

On the System Options tab, Affiliate System, Station Interface, and Split Copy must be enabled.

Traffic: Vehicles -> Options -> General

Make sure "Allow Split Network/Copy" is set to Yes, and define the time zone feeds.

Affiliate Stations -> Interface

Serial #1 and Serial #2 are used if there is one station that receives two commercials.  For example if there is an AM/FM combo, one station may use serial port #1 for its split copy and the other station would use the other port. 

Affiliat.ini

Should you need the audio extension to be different than mp2, you can enter the correct audio extension name in the Affiliat.ini file.


The audio extension can be any of the following: Xxx, “Xxx”, .Xxx, or “.Xxx”. The extension will appear lowercase, unless it is part of the file name or caption. 

Running the Export

  1. Export Start Date: Enter the start date of the export 
  2. Number of Days: Enter the number of days to be exported
  3. Export All Spots or Regional Spots: Decide whether to send all spots or only those with split/regional copy
  4. Run Letter: This is the last character in the file name, and allows multiple runs to be distinguished from one another
  5. Export to Folder: Select the folder to which you want to save your Files (generally CSI\Prod\Export)
  6. Select the Vehicles and Stations to Export and click the Export button.
