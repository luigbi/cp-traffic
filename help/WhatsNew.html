What's New

Individual login accounts 

Each user will have their own individual account and will log in using their email address and password instead of logging in using call letters and a shared password. This enhancement has been added for security purposes.

Network Admin Accounts

Network personnel can now log into the Affidavit website using a network admin account. Network admin accounts can be used to add personnel (both network admins and station personnel), configure system settings, and view/post station information.

Creating New Accounts

When a new personnel account is now created, an automated email will be sent to the new user with a link to verify their email address and to create their password. The new user must click the link within 24 hours to complete the process of creating a new account. 

Language Options 

It's now possible to switch between different languages by clicking the language toggle in the upper right corner. The current available languages are English and Spanish.

Dashboard 

When logging in, users will be taken to the main Dashboard page, which lists the available logs and affidavits. At any point, clicking the Dashboard button in the upper left corner will take users back to the Dashboard page. 

Printing Logs 

When printing a log, users can now choose whether to print to a printer or whether to print to a PDF file by selecting the chosen output from the Printer dropdown on the Print page. 

Selecting Days and Weeks to View and Post 

A new pop up appears that is used to select individual days or weeks to view/post for the Log and Affidavit. 

Citation Page 

The first and last name of the logged in user is automatically inserted into the name field on the Citation page, saving time and increasing accuracy. Station users can simply check on "I agree" and press the "Continue" button to agree to the citation and access the Affidavit. 

Affidavit Page 

There's now a dropdown menu to select whether to view "All" spots, "Posted" spots only, or "Unposted" posted only. The default is "All". 

Help Page 

Users can access the Counterpoint Web Affidavit Portal Manual by clicking the Question Mark in the upper right corner of every page.
