Sports Check Programs

If you are using Live Log with Events, you may have some dates not defined as complete.  You cannot invoice any date that has not been set as complete.

Log Posting Status Report

The Log Posting Status Report can tell you if there are any “Day is Complete Discrepancies”.  


You may also use an automation system to import spots. This report can show you the status of the dates before attempting to run final invoicing.


The Day is Complete Status will be shown for the vehicle/date selected (as a sub-header). There are 3 statuses that show: 


  • Complete: Day has been imported or manually flagged as Complete.
  • Incomplete: The day has been exported and no import has taken place, or a manual posting has been done and the complete flag has not been set, or Live Log logger did not set Event as Complete.
  • Day Not Posted: No automation export, importing or manual posting has been done.  When a vehicle/day has been exported by Automation, the Day is Complete flag is set to "Incomplete".  When that vehicle/day is imported, the Day is Complete flag is set to "Complete".  When a vehicle/Event has been posted due to live log, the logger will sign off as completed or not. If Completed, the Day is Complete flag is set; otherwise the vehicle/day/Event remains "Incomplete"


Note: The Day Is Complete [X] flag on the Post Log Screen is not available for vehicles reconciled via the automation import.

Game Data Verification

Sport Contract Check

The Sport Contract Check is used for these purposes:

  • To compare # spots selected for sports vehicle on a contract and the number of spots associated with line flights by week. The two values should match
  • Check that the Event dates match between the contract and the program definitions for the Event


The User can choose to let program automatically fix above problems.


To run the check program select the menu item Accessories -> Utilities -> Sport Contract Check


Enter a Monday Start Date and leave Check defaulted on to check for problems.



If any errors are found, it will appear in the results area at the bottom of the screen.


To fix problems, first select the ‘fix’ radio button, then press Fix.


The Utility creates and stores the SportChk.csv in the CSI\prod\data\messages folder.
