Copy Issue Troubleshooting

This section covers different troubleshooting tips that can be used to help determine what went wrong in various copy related scenarios.

Wrong inventory assigned to vehicle

If the wrong ISCI or cart number is assigned to a vehicle and is appearing on the Traffic log or Traffic export, run the Copy Rotations by Advertiser report for the problematic advertiser to see what rotations exist and what copy is being picked up by the vehicle. Even if the correct copy was entered, there may be reasons that explain why the incorrect copy is showing on the log as a result of the way the rotations were entered.

  • Find the rotation for the correct copy on the report. If it’s not on the report or there’s a higher rotation number that is superseding it, the rotation may need to be entered again.
  • If the rotation is on the report and looks correct, check the start and end date, start and end times, and valid days for the rotation to make sure they’re covering the spots in question. For example, if the spot is on Saturday and the valid rotation days are Monday through Friday, the Saturday spot will not pick up the copy from that rotation.

Wrong copy is showing on the affiliate web log

This may be due to making a copy change after the logs were exported to the web site. To resolve:  

  • Check to see that the correct copy rotations are entered into Copy screen. The Copy Rotations by Advertiser can be used to assess this, or the copy rotations can be viewed from the Copy Rotation dropdown.
  • If the correct copy was not entered, enter it now.
  • Re-generate the logs and assign copy so that the correct copy is showing in the Affiliate System.
  • Re-export the web log using the Counterpoint Affidavit export.

Generic copy appearing for a station instead of regional

If the copy assigned for a station is the generic copy when it should be getting the regional copy, first run the Copy Rotations by Advertiser report and find the generic and regional rotations. 

  • Check to make sure the generic ISCI is in a rotation with a number lower than the regional rotations. 
  • Check the assign dates. Has the rotation been assigned? 
  • If everything looks correct on the Copy Rotations report, run the Copy Regions report to check to see if the station is actually part of the region used for the regional copy rotation. 
  • Check the station information defined on the Affiliate System Stations screen to verify that the DMA, MSA, Format, State and Time Zone are defined correctly.

Regional copy appearing for a station instead of generic

If copy from an incorrect region is assigned to an affiliate, run the Copy Rotations by Advertiser report first to see what generic and regional rotations have been entered. Then run the Regions report to check and see if the station is actually part of a region that it should not be part of.
