Vendor Setup

The Vendor Setup screen is accessed from the Affiliate System File menu.


The Vendor Setup screen is used to indicate whether various log delivery services and audio delivery services are active or dormant, and for storing connection information for some services. A keycode is required for updating the vendor list. 



The log and audio delivery services are: CBS, Stratus, IDC, iHeart, Mr. Master, Network Connect by Marketron, RadioTraffic.com, RCS, Synchronicity, Wegener-Compel, Wegener-iPump, WO Traffic Radio Interchange, X-Digital Break and X-Digital ISCI. (Not all of these services are implemented as some are part of future enhancements.) 


To use one of these services, the status must be set to Active on this screen. 


The Vendor Type, Delivery Method, Web-to-Vendor, Vendor-to-Web, and Hierarchy settings are all pre-determined and cannot be changed.


Additional settings are available for each service:


Allow Auto-Posting: When checked on, spots that are imported from a vendor service to the Counterpoint Affidavit website will be automatically posted as they are imported.


Allow vendor to update delivery status: This setting is only available with the Mr. Master service and it allows web version 2 stations to be posted by Mr. Master.


Allow stations to override vendor times: If this is checked on, station users can override the air times of spots imported from this vendor service. If not checked on, then stations users cannot override those imported air times.


Vendor and Station Connection Info: If the service is a csiStationService, then the Station Connection Info must be defined. If the service is a csiVendorService, then the Vendor Connection Info must be defined. Wide Orbit requires both the Vendor and Station Connection Info to be defined.
