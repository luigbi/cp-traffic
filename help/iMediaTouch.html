Replace “COM” with media code

In Traffic Site Options, on the Automation tab, there’s a checkbox next to the iMedia Touch export labeled “replace COM with media code”. When checked on, the hardcoded “COM” indicator that starts in column 27 of the output will be replaced with a three letter media code from the Media Definitions screen, Description field.


Example with “COM”:

09:31:30 00:30 ZM069      COM              Walmart/Clothing         22098 


Example with “COM” replaced by media code “AAA”:

09:31:30 00:30 ZM069      AAA              Walmart/Clothing         22098 


When this feature is enabled, if copy has not been assigned for a spot, as there is no media code to use, COM will be used instead. If the media code description is greater than three characters, it will be truncated to three.
