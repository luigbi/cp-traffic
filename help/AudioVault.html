Audio Vault RPS   

Export and Import

Variable Length, vertical bar delimited, Filename: SSSSSDDD (no extension) 

All records terminated with c/r l/f. ( SSSSS = station code, DDD = 3 char day of week).  


In the Programming Library screen, the user creates an Event type named RPS Comment. Only this Event type name will be recognized in the export to create commands. In the comments field, enter text for each line required in the AV export, separated by a semicolon (“;”). If Audio Vault RPS is a selected automation export, vertical lines and semicolons become allowable special characters. 



For example:   comment field from programming screen-
12:09:28PM RPS Comment                                +|Indicator5||X;+|ID05||
Generates 2 records in AV RPS export-
|3:19:28PM|+|Indicator5||X
|3:19:28PM|+|ID05||

Every RPS comment event created in the AV export will contain vertical line, time, vertical line, followed by the comment.
|06:00:00PM|Comment here

Spots will contain vertical line, time, plus sign, vertical line, cart # followed by 2 vertical lines, Advertiser Name and internal spot code.
|06:00:00PM|+|1234||AdvertiserName      12345678

The RPS comments will show on the L73 log.  


On the Audio Vault RPS export, immediately before the line for any spot with split copy, the "Audio Vault Indicator ID" from the advertiser split cue screen will be output, using the Audio Vault Indicator ID for the advertiser with the split copy.


The following is a partial example of the Audio Vault RPS export. The Indicator ID defined on the Split Cue screen for advertisers AC Delco (with regional copy) is "Indicator1||X". (The time and |+| gets output automatically by the export program.)

|09:00:00PM|+|Indicator1||X

|09:00:00PM|+|074||AC Delco 22969


Because of this feature, the split cue indicators for split copy should not be entered in the Programming screen.


Important Note: the Audio Vault Indicator ID has been made to work with both the HB and HBP X-Digital avail form. However, when using the HB form, the split copy must be in the first position of the break (and only in the first position). If the HBP avail form is used, the split copy can be placed in any position of the break.
