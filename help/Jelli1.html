Jelli

The Jelli export file is created when the Counterpoint Affidavit export is generated if the Jelli export feature is turned on.

Setup

There are three areas in the system that must be activated for the export to work. 

Traffic Site Options

Enable the export by checking on Jelli in Traffic -> Site Options -> Options. You will need a KeyCode from support@counterpoint.net to do so. 

Traffic Vehicle Options

In the Traffic system, go to the Vehicles -> Options -> Export tab, and check on Jelli for each vehicle that will need to be part of the export. 

Affiliate Agreements

In the Affiliate System, each agreement that is to be included in the export must have Jelli selected as a Delivery method in the Agreements>Delivery tab.

Running the Export

The export file will be generated automatically (for those vehicles defined to be part of the export) during the Counterpoint Affidavit Export. The Jelli file is saved in the CSI\Prod\Export folder.

File Name

The naming structure is Jelli – MM- DD-YYYYA, where Jelli- is hardcoded, and followed by the start date of the export.


If the export is generated more than once for the same start date, the following export files will have a letter appended to the name, i.e., Jelli-121415A. 

File Content

The comma delimited file contains one record for every spot. Alpha fields will be enclosed by double quotes, so that commas can be allowed within a field.


       Sample Record : "PETW2007",2012-05-14 11:45:00,30,"Live for Now","KYLI", “FM”, “XYZ Network”, 12345, 06:00:00, 10:00:00, 1, 1, 1, 1, 1, 0, 0, 1002,"PEPSI","OMD",297
