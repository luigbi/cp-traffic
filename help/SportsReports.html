Sports Reports

Below are some key reports that are tailored to help you manage your sports inventory and event reconciliation.

Event and Sports Avails Report

The Event Avails report lists all inventory and avails per event. The report can be run in a detail or summary mode by choosing to show or not show avail names. 


If Show Avail names is checked, the detail version shows of the report will display one line per avail name, listing the inventory, sold, average :30 spot rate, available inventory, % of inventory sold, and % of available inventory.  Game totals are summarized after each game.


If you choose to not Show Avail Names, the report summarizes the totals for each game on one line.

Special Event and Sports Avails Export

On version 8.1 and above, an additional report destination option of "Export" is available on the Event and Sports report that when selected will export the report data directly to an Excel file, and include some additional information not normally shown on the regular report. This new export option will display one row of data for each event, with additional data including normalized 30s data for inventory, sold, and remaining avails for each event. There are also sections for Drop-in, Billboard, and Wegener-related inventory. The avails that are grouped together in these different sections are defined on the Avail Names screen. The report selection screen also changes to allow the special Drop-in, Billboard, and Extra sections to be included or excluded from the output. Important note: Microsoft Excel is required to be installed on the target system in order to run this special version of this report.

Event Summary

The Event Summary Report can sort by vehicle or date.  It is a listing of all the Event information for each Event Schedule.   

Spots by Date and Time

The Spots by Date and Time report can show you the spot placement per event.


The report shows the following details:

  • Shows Event information in the header
  • Subtotals spot counts and revenue for each individual Event within a day
  • Gross or Net spot revenue 
  • When a spot crosses midnight, the report will show the true date where the spot. For example, an Event starts at 9PM on Sunday (5/28) and runs overtime into Monday (5/29) morning.  
    • The spots are scheduled as:
      • Sunday,5/28    10:45PM
      • Sunday,5/28    11:30PM
      • Sunday,5/28    12:30AM
      • The report will adjust the spot at 12:30AM to be 5/29


Locked Avails  

The Locked Avail Report lists the vehicles, days and times that have been locked/pre-empted by an event.  The vehicles are automatically locked if you are using the Pre-empt feature and you have booked your sport event with a firm date and time. If the sports event times are changed, the locked avails are adjusted.


Live Log Activity 

The Live Log Activity Report shows all games posted via Live Log. The report lists all games posted, as well as who posted them and at what times. . If a game does not show up on the Live Log activity report, follow-up with the Board Ops as to why it was not posted. If the game did air but was not posted, you can manually post the event in Post Log.

Log Posting Status

The Log Posting Status is an excellent tool to determine which events has been posted via Live Log. If an event has been posted, it will be listed as Day is Complete.

Proposal/Order

When entering the schedule line for a sports buy, there is an option to show the Events on the order.

Revenue on the Books

Revenue on the books lists all revenue on a Vehicle(s), by advertiser and contract. When generating the report with Vehicles as the Sort #1, there is an option to break out revenue by event, or combine the dollars for the entire vehicle. If an advertiser crosses seasons, the revenue must be broken out by individual event.
