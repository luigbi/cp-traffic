Month End Check Procedures

Before generating Final Invoices, there are a few reports and check programs that should always be reviewed:

  • Paperwork Summary
  • Contracts Missing Copy
  • Missing ISCI Codes
  • Missed Spots Report
  • Spot Discrepancy
  • SSF Check
  • SMF Check
  • CFF Check
  • Sports Contract Check

Paperwork Summary

The Paperwork Summary Report will show all scheduled Orders and Holds, as well as any Working Proposals for a week. You can use this report to ensure that all Proposals that are ready to become Orders are changed to an Unscheduled Order and schedule them.


To generate:

  • Go to the report screen and double click on Paperwork Summary.
  • Enter the date span you wish to generate (Standard Broadcast month to be billed)
  • Select All Advertisers.
  • To see Proposals that are being Worked on/Revised, you must include Working.
  • To see Proposals that have not yet been approved as orders, check Unapproved.
  • Leave other selections defaulted.
  • Click the Generate Report button.

Contracts Missing Copy

The Contracts Missing Copy Report is used to check if all spots have copy. There is a legend along the bottom of the report that explains how to read the report. If a spot is listed as missing copy, make sure you create a rotation prior to generating invoices.


To generate:

  1. Go to the report screen and double click on Contracts Missing Copy.
  2. Enter the date span you wish to generate.
  3. Select All Vehicles.
  4. Leave other selections defaulted.
  5. Click the Generate Report button.


If time zones are defined Vehicles -> Options -> General tab, Contracts Missing Copy verifies that all spots have copy rotations for the requested date span plus one day to ensure that any spots coming from Eastern time zone fed avails or selling avails that cross midnight and are actually airing within the requested date span have rotations.

 

For example, if a 10pm Pacific Time Sunday spot is actually coming from 1am Eastern avail on Monday, the 1am Eastern avail will be included on the report since it is airing within the requested date span.


Selling vehicles are treated the same. The system will look at the times of the selling and airing links, and verify the appropriate selling avails have rotations. Here are two examples of how this will work: 

  • A 4am selling avail that links to a 2am airing avail: the 4am selling avail will be included on the report since it is airing within the requested date span.
  • A 2am selling avail that links to a 4am airing avail: the 2am selling avail will not be included because it is not within the requested date span.

Missing ISCI Codes Report

If you move spots in the past in Post Log, or have a contract revision that affects spots in the past, there is a chance that the contract will have a rotation that does cover the spot moves. Missing ISCI Codes will ensure every spot has a valid ISCI code. The report checks the ISCI requirements of the Advertiser and Agency, to verify that all spots requiring ISCI codes have valid ISCI codes assigned. Any spot missing and ISCI that requires one, will appear in red.

Missed Spots Report

Generating the Missed Spots report on a weekly basis will ensure that all spots will be billed and transferred to the website. If missed spots are resolved weekly, the report should come up blank at the end of the month.


To generate:

  1. Go to the report screen and double click on Missed Spots.
  2. Enter the date span you wish to generate.
  3. Select All Vehicles.
  4. Leave other selections defaulted.
  5. Click the Generate Report button.


The report should come up blank. If spots do appear, reconcile them in Post Log.

Spot Discrepancies

The Spot Discrepancies report checks that all spots ordered were created and scheduled according to the contract parameters.


To run the spot discrepancy report:

  1. Go to the Report screen and double click on the Spot Discrepancies report.
  2. Enter the start and end date of the entire month.
  3. Leave All Advertisers selected.
  4. Click Generate.


If there are any discrepancies, email support@counterpoint.net.

Log Posting Status 

If the option “Day is complete testing” is checked in Site Options -> Invoicing, M-Su check boxes appear in Post Log. Each day must be checked for each vehicle prior to running invoices. If not checked, you will receive a warning message when trying to run invoices. 


To verify all days are checked, run the Log Posting Status for Unbilled spots with the “day is complete summary” radio button selected and the “day is complete discrepancies only” checkbox checked on. 

SSF Check

SSF Check ensures the integrity of each spot created.


To run the SSF Check:

  1. Click on Accessories in the upper left corner of the Jobs screen.
  2. Select Utilities, then SSF Check.
  3. Enter the start and end of the month for the start and end dates.
  4. Leave Check defaulted.
  5. Select All Vehicles. 
  6. Click the Check button.


The white results box will list every vehicle, and should say 0 Added, 0 removed. If any errors appear, email support@counteproint.net.

SMF Check

The SMF Check Utility checks the integrity of the scheduled spots against the missed spots. To run the utility:


  1. Select the Check/Fix button.
  2. Verify the results of the check via Accessories -> Messages -> SMF Check.
  3. If there are any discrepancies that show in the white box, email the SMFCheck.txt to support@counterpoint.net

CFF Check

This verifies schedule lines and weekly flights consistent. Run it in Check mode if any messages are displayed please email them to Counterpoint, These message will be saved in the csi\prod\data\messages folder, then run the CFF check in Fix mode.

 

  1. Select the Check option.
  2. Define the billing start and end dates.
  3. Select the Check Button.  
  4. If there are any discrepancies that show in the white box, email the CffCheck.txt to support@counterpoint.net.

Sports Contract Check

If game vehicles are used, the Sports Contract Check utility can be run to verify that the Game schedule defined in programming and the game dates retained with the contract match.  

  1. Select the Check option.
  2. Define the billing start and end dates.
  3. Select the Check Button.
  4. If there are any discrepancies that show in the white box, email the SportChk.csv to support@counterpoint.net
