Dashboard Page

After signing on, the Dashboard will be displayed. The Dashboard is the main menu and is used to navigate to the logs and affidavits, view a count of the overdue weeks, create makegoods, and access the Personnel page.


When signed on as a Network Admin (and only when signed on as a Network Admin), as shown in the picture below, a Settings button and Station dropdown menu will be available at the top of the screen.



Network Admins can switch to the data for a different station by selecting it from the station dropdown. You can also type call letters into the Search window to search the list of stations, as shown in the picture below.



Station users will not have a Settings button. The station dropdown will only be visible for them if their email address is associated with more than one station. 


This picture shows an example of what a typical station user will see on the Dashboard page.



At the top of the screen, on all pages, the language control can be used to switch between the available languages. The question mark button can be used to access the help system. After logging in, the user profile in the upper right corner can be clicked on to log out.
