Station Information

The Station Information export creates a CSV file containing specific station information.  It creates ‘StationInformation.csv’ in the CSI\Prod\Exports folder and a log file ‘StationInformation.txt’ in the Data/Messages folder.


There are 4 options that are not exclusive and can be combined: Agreements, X-Digital, Wegener, and OLA.


Select the information you wish to export by checking the corresponding checkbox and then click the Export button to run the export.

Export Contents

The first line states what the file is: “Counterpoint Station Information Export”


The second line is column headings: “Call Letters,City of License,Market,Market Code,Market Rank, Format,Format Code,State Name,ST,State Code,Zone,Zone Code,Owner,Agreements,X-Digital,Wegener,OLA.”

 

The third line begins the exported data. All lines following this line are data.  


The data is formatted as shown below.  A ‘Y’ in ‘Comma-Delimited’ means the column value  may have a comma in it, therefore it is wrapped in quotes.  An ‘N’ means commas are never returned within that field. No quotes are returned.


The final 4 fields return an ‘x’ for “true” and blank for “false”.
